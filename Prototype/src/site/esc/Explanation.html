This is just a stupid example of what an Escalator document might look like. This documentation is basically a Markdown document, with some extensions (indicated by an initial // on a line) that are caught by our preprocessor to handle extracted code fragments and embedded tests/examples. Here is the code for a factorial function: 

/**
 * A non-OO object to compute factorial.  It's not good FP either...
 */
object Factorial {
  def apply(n: Int): BigInt = if (n <= 1) 1 else Factorial(n - 1) * n
}

Now we can test it:

Factorial(7) must_==(5040)

  + Factorial(7) is 5040
